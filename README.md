# Careermap-india
An NLP-powered analytics assistant over India's regional unemployment data. Ask plain-English questions ("Which state has the highest unemployment?", "Compare Gujarat and Maharashtra", "Predict unemployment in Gujarat") and get a data-grounded answer, a chart, and a map — all computed locally with pandas / NumPy / scikit-learn / Plotly. No LLM, no external API.

Features
Natural Language Querying: Talk to your data directly.
Interactive Visualizations: Dynamic charts and mapping using Plotly.
Predictive Modeling: Simple scikit-learn forecasting for unemployment trends.
Local Gradio Interface: Elegant web UI that runs entirely on your local machine.
How to Run
Install dependencies:
pip install pandas numpy plotly gradio scikit-learn
Navigate to the project directory:
cd unemployment
Run the application:
python app.py
